## Overview

This project explored the use of **multi-agent reinforcement learning** concepts to detect and counter the spread of misinformation. The system was designed as a swarm of agents, each contributing independent signals to help identify potentially misleading or harmful content.

The focus of the project was on **conceptual system design and rapid experimentation**, rather than building a fully production-ready solution.

---

## High-Level Idea

- Multiple agents operate in parallel
- Each agent analyses information from different perspectives or data sources
- Agents collectively contribute to identifying misinformation patterns
- A swarm-based approach is used to improve robustness compared to single-model decisions
